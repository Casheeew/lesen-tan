(examples use German (de) to English (en))
   
1. create a .env file based on .env.example

2. if your language is not in languages.json, add it

3. run ./auto.sh German English

4. if everything runs, dictionaries should be in data/language/de/en

5. (optional) data/language/de should contain files with skipped tags for ipa and terms,
   you may want to add some to tag_bank_ipa.json or tag_bank_term.json and rerun ./auto.sh

6. import the zipped dictionaries into yomitan in your browser
